ATM project

ATM project started as a project for one of the JAVA modules in Baltic Talents academy. Team consisted of myself and Mykolas Pinkevičius and we worked under SCRUM principles. My main objective for this project was to implement a way where a user would be able to authenticate using a card where authentication data would only be kept with the user and not on the database or within the code. I achieved this by employing RFID technology, NFC enabled device, REST and PostgreSQL.
